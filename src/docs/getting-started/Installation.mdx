  ## Install the package

  ```bash
  npm install @apicenterlabs/shockwave
  ```

  <Note title="Peer dependencies">
    The consuming application should provide <code>react</code> and <code>react-dom</code>. Shockwave supports React 18 and React 19 consumers.
  </Note>

  ## Start using components

  ```jsx
  import { Button } from '@apicenterlabs/shockwave';

  export default function App() {
    return <Button variant="primary">Launch flow</Button>;
  }
  ```

  ## Styles

  Styles are loaded automatically from the package entry in typical bundler-based apps. If your app prefers explicit stylesheet control, you can import the generated stylesheet yourself.

  ```jsx
  import '@apicenterlabs/shockwave/style.css';
  ```

  ## Verification checklist

  - Install the package in a React app.
  - Import a component from `@apicenterlabs/shockwave`.
  - Confirm the component renders with styles applied.
  - Use Storybook as the reference for props, variants, and system guidance.
