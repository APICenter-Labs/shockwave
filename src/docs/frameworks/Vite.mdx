  ## Install

  ```bash
  npm install @apicenterlabs/shockwave
  ```

  ## Example

  ```jsx
  import { Button } from '@apicenterlabs/shockwave';

  export default function App() {
    return <Button variant="primary">Open dashboard</Button>;
  }
  ```

  <CardGrid>
    <Card title="Peer dependencies">
      <p>Your Vite app should already provide <code>react</code> and <code>react-dom</code>. No extra framework adapters are required.</p>
    </Card>
    <Card title="Styles">
      <p>Shockwave styles load automatically with the package entry, but you can import <code>@apicenterlabs/shockwave/style.css</code> explicitly if your app prefers that pattern.</p>

  <Note title="When to use this setup">
