  ## Install

  ```bash
  npm install @apicenterlabs/shockwave
  ```

  ## Example

  ```jsx
  import { Button } from '@apicenterlabs/shockwave';

  export default function App() {
    return <Button variant="primary">Continue</Button>;
  }
  ```

  <CardGrid>
