  ## Install

  ```bash
  npm install @apicenterlabs/shockwave
  ```

  ## Example client component

  ```jsx
  'use client';

  import { Button } from '@apicenterlabs/shockwave';

  export function DashboardAction() {
    return <Button variant="primary">Create workspace</Button>;
  }
  ```

  <Note title="Client components">
    If the place you render Shockwave needs event handlers, mark that file as a client component in the Next.js app router. Purely static rendering can stay server-side.
  </Note>
