# ⚡ Shockwave

**Shockwave** is a modern, reusable React UI component library built for speed, consistency, and scalability.

Built and maintained by **APICenter Labs**.

---

## 🚀 Features

* ⚡ Lightweight and fast
* 🧱 Reusable components
* 🎨 Simple and clean design system
* 📦 Easy to integrate in any React app
* 🔧 Built with Vite for optimal performance

---

## 📦 Installation

```bash
npm install @apicenterlabs/shockwave
```

---

## 🔥 Usage

```jsx
import { Button } from "@apicenterlabs/shockwave";

function App() {
  return <Button variant="primary">Click Me</Button>;
}
```
